# Wedding Guest Manager - Senac Wedding

Sistema para controle de acesso em tempo real para o evento em Cambará do Sul.

## Tecnologias
- **Backend:** Node.js com Express
- **Frontend:** HTML5, Tailwind CSS, jQuery (Ajax)
- **Banco de Dados:** MySQL

## Como rodar
1. Importe o arquivo SQL fornecido para seu MySQL.
2. Na pasta `server/`, execute `npm install`.
3. Inicie o servidor com `node app.js`.
4. Abra o arquivo `client/index.html` no navegador.
